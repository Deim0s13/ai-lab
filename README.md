# AI Workstation Lab

Personal AI-augmented architecture workstation project.

This repo contains local tools, context files, and documentation for building a GPU-backed AI workstation using:

- WSL2 Ubuntu
- Ollama
- Podman
- Open WebUI
- VS Code / Continue.dev
- local LLMs
- CLI-based architecture workflows

## Core Tools

| Tool | Purpose |
|---|---|
| `ask-ollama` | General local AI CLI interface |
| `architect-ai` | Architecture-focused workflow wrapper |
| `ai-status` | AI workstation health check and safe remediation |
| `ai-gpu-check` | GPU acceleration validation |
| `ollama-api-test` | Direct Ollama API testing |

## Context Files

| File | Purpose |
|---|---|
| `architect.txt` | Architecture persona |
| `writing-style.txt` | Preferred writing tone and style |
| `banking.txt` | Banking / financial services context |
